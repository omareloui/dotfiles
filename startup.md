# Startup Scripts I use

Run `Startup Applications` and add the scripts. Or add them through `gnome-tweaks`.

- Start Telegram to tray

  ```bash
  telegram-desktop -startintray -- %u
  ```

- CopyQ

  ```bash
  "/usr/bin/copyq"
  ```

- ULauncher

  ```bash
  env GDK_BACKEND=x11 /usr/bin/ulauncher --hide-window
  ```
